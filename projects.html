Assignment 3
The program is meant to log "One Rep Max" calculations for one set of a workout, along with the workout name, energy level felt, and date. This "Oner Rep Max" calculation represents the approximate maxiumum weight that the person could lift with one rep for that workout. To log a workout set, press the "Log a Weight Set" button on the home screen. Next there will be information to enter regarding workout name, weight, reps, and energy level. Press "Save" to log the data. Optionally, one can just press the "Log" button to view the log. Log entries can be deleted by swiping on them to the left and pressing the "Delete" button.

Assignment 2

This is a slightly more complex version of the one rep max calculator from assignment 1. The main differences are that it now makes use of a navigation controller, and a table view controller has been added. The main functional difference is the addition of the 'Log' button. When the 'Log' button is pressed, the current calculated 'one rep max' is stored to an array, and when the user navigates to the History page, the array is segue'ed to the history view controller and appended to the list. The History page shows a list of all the logged one rep max entries.
